Nginx Unit初探及其性能对比分析

Nginx Unit是一个动态的网络应用服务器，其设计初衷就是可运行多种编程语言；
Nginx Unit使用REST API进行动态配置；没有静态配置文件，所有配置更改直接在内存中发生且无需重新加载配置或重启服务即可生效。
它具有以下特征：

  • 使用RESTful JSON API进行动态重配置服务器；
  • 同时支持多语言，多版本应用；
    即将支持的特性有：
  • 动态语言的进程管理功能；
  • TLS支持；
  • TCP,HTTP,HTTPS,HTTP/2路由和代理。

下面我们以一个简单的php-test应用为例介绍unit的安装和使用。
